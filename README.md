

## About 

- Backend: Laravel
- Frontend: Vue.js
- Database: Mysql

## Installation

- ```git clone https://github.com/hovhannestsaturyan/AdCash.git```
- ```cp .env.example .env```
- ```composer install```
- ```php artisan migrate```
- ```cd client```
- ```cp .env.example .env```
- ```npm install```

## Run Project

- ```Laravel: php artisan serve```
- ```Vue: npm run dev```
